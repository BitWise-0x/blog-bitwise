# Security Policy

## Supported Versions

Only the latest version deployed from the `main` branch is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Email:** Send details to [rob@bitwisesolutions.co](mailto:rob@bitwisesolutions.co)
2. **GitHub Security Advisories:** You can also use [GitHub's private vulnerability reporting](https://github.com/BitWise-0x/blog-bitwise/security/advisories/new)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Resolution:** Assessed on a case-by-case basis depending on severity

Please do not open public issues for security vulnerabilities.
